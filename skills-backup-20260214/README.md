# Oskris Web Development Skills Suite

专为网站开发接单业务设计的完整 AI Skills 工作流程系统

## 📋 概述

这套 Skills 将原本的"多个 Agent"架构转变为"1个 Agent + 4个专业 Skills"的高效系统，专注于马来西亚网站开发接单业务。

## 🎯 核心 Skills

### 1. requirements-analyst (需求分析师)
**作用**：系统化收集和分析客户需求

**何时使用**：
- 首次客户接触
- 项目发现阶段
- 范围定义
- 预算评估

**输出**：
- 完整的需求文档
- 技术规格
- 预算估算
- 项目时间表

**关键特点**：
- 8个核心问题框架
- 马来西亚本地化考虑
- 风险识别
- 客户期望管理

---

### 2. design-consultant (设计顾问)
**作用**：将需求转化为视觉设计方案

**何时使用**：
- 创建设计提案
- 选择配色方案
- 规划布局结构
- 向客户展示设计概念

**输出**：
- 设计提案文档
- 配色方案
- 布局规划
- 组件设计

**关键特点**：
- 针对不同行业的设计风格
- 马来西亚文化适配
- 移动优先设计
- 完整的设计决策框架

---

### 3. frontend-builder (前端开发)
**作用**：将设计实现为可用的网站

**何时使用**：
- 实施批准的设计
- 构建网站布局
- 开发交互功能
- 代码审查和优化

**输出**：
- 生产就绪的代码
- 响应式网站
- 性能优化
- 技术文档

**关键特点**：
- 完整的技术栈建议
- WordPress 专业开发
- 马来西亚支付集成
- SEO 最佳实践
- 完整的组件库

---

**作用**：统筹整个项目从开始到交付

**输出**：
- 项目计划

**关键特点**：
- 9个阶段完整流程
- 客户沟通模板

- 完整的项目检查清单

## 💡 使用建议

### 对于新项目：

1. **先用 project-workflow**
   - 了解整体流程
   - 准备必要工具

2. **客户接触时用 requirements-analyst**
   - 系统化收集需求
   - 创建需求文档

3. **设计阶段用 design-consultant**
   - 基于需求创建设计
   - 获得客户批准

4. **开发阶段用 frontend-builder**
   - 实施设计
   - 编写高质量代码

5. **全程由 project-workflow 统筹**
   - 管理时间线
   - 客户沟通
   - 项目交付

### 对于现有技能：

这些 Skills 可以与你现有的知识结合：
- iOS 开发经验 → 理解移动优先
- Kong Kiong 项目 → 作为 Portfolio
- 硬件店背景 → 理解本地商家需求
