本目录包含 Kris 网页开发接单业务的专业 Skills 套件。

```
skills/

│   ├── design-consultant/      # 设计顾问 Skill
```

## 🎯 核心 Skills（business-workflow/）

### 完整的网页开发接单工作流

这4个 Skills 形成完整的项目执行链：

**1. requirements-analyst** - 需求分析师
- 📋 收集客户需求
- 💰 预算评估
- 📊 技术规格
- ⏱️ 项目时间表

**2. design-consultant** - 设计顾问
- 🎨 视觉设计方案
- 🌈 配色选择
- 📱 布局规划
- 🖼️ 组件设计

**3. frontend-builder** - 前端开发
- 💻 代码实现
- 📱 响应式设计
- ⚡ 性能优化
- 📝 技术文档

**4. project-workflow** - 项目管理
- 📅 进度管理
- 💬 客户沟通
- 📦 项目交付
- 🎯 质量控制

## 🚀 快速开始

### 新项目工作流

```
1. 客户咨询
   ↓
2. 使用 requirements-analyst 收集需求
   ↓
3. 创建提案和报价
   ↓
4. 使用 design-consultant 制定设计方案
   ↓
5. 使用 frontend-builder 开发实现
   ↓
6. 使用 project-workflow 统筹全程
   ↓
7. 测试、部署、交付
```
