## Phase 1: Lead Generation

### Marketing Channels:

**Local Outreach**:
- Visit nearby businesses
- Network at business events
- Referrals from existing clients
- Partner with Kong Kiong customers

**Online Presence**:
- Oskris.com portfolio
- Facebook Business Page
- Instagram showcase
- Google My Business
- LinkedIn profile

**Paid Advertising**:
- Facebook Ads (RM200-500/month)
- Google Ads (RM300-800/month)
- Instagram promotions

### Lead Qualification Questions:

Before spending time:
1. Do they have a budget? (Minimum RM3,000)
2. Do they have decision-making authority?
3. Is timeline realistic? (Minimum 4 weeks)
4. Are they comparing multiple quotes?
5. Do they understand what a website involves?

**Red Flags** (consider declining):
- "I only have RM500 budget"
- "I need it done by next week"
- "Can you work for free and take payment from sales?"
- "My nephew can do it cheaper"
- Constantly changing requirements
- Refuses to pay deposit

## Phase 2: Initial Consultation

**Objective**: Understand client needs and gather requirements

**Duration**: 1-2 hours

**Location**: Client's office/shop (builds trust) or video call

**Process**:
1. Use **requirements-analyst** skill
2. Ask all 8 core question sections
3. Take notes and photos (if physical visit)
4. Explain your process
5. Set expectations

**Deliverable**: Requirements document

**Timeline**: Within 24-48 hours after meeting

## Phase 4: Design Phase

**Duration**: 1-2 weeks

**Process**:
1. Use **design-consultant** skill
2. Create wireframes
3. Present to client
4. Get feedback
5. Create full mockups
6. Get approval

**Client Touchpoints**:
- **Week 1**: Present wireframes
- **Week 2**: Present mockups
- **Revisions**: As needed (max 2 rounds included)

**Deliverables**:
- Wireframes (low-fidelity)
- Design mockups (high-fidelity)
- Design specifications
- Asset files

**Client Approval Required**: Yes (in writing)

## Phase 5: Development Phase

**Duration**: 2-4 weeks (depending on complexity)

**Process**:
1. Use **frontend-builder** skill
2. Set up development environment
3. Build according to approved designs
4. Weekly progress updates to client

**Week-by-Week Plan**:

**Week 1**:
- Set up hosting/domain
- Install WordPress (if applicable)
- Build homepage
- Build basic structure

**Week 2**:
- Build inner pages
- Add content
- Implement features

**Week 3**:
- Connect forms
- Add payment gateway (if e-commerce)
- Implement SEO

**Week 4**:
- Testing and bug fixes
- Performance optimization
- Mobile testing

**Client Touchpoints**:
- Weekly progress screenshots/videos
- Demo link for review
- Feedback collection

**Deliverables**:
- Staging website (for client review)
- All pages completed
- All features functional

## Handling Difficult Situations

### Situation 1: Client Wants Major Changes After Approval

**Response**:
"I understand you'd like to make these changes. Since we've already completed the approved design phase, these would be considered additional revisions. The cost for this scope change would be RM [X]. Would you like me to prepare a quote?"

### Situation 2: Client Delays Providing Content

**Response**:
"Hi [Client], we're ready for your content to continue development. To keep us on track for your [Launch Date], I'll need the content by [Date]. Otherwise, we may need to push the launch date back. Let me know if you need help with content creation (additional RM [X])."

### Situation 3: Client Wants Free Extra Features

**Response**:
"That's a great idea! This feature wasn't included in the original scope, so it would be an add-on of RM [X]. Would you like me to add it to the project?"

### Situation 4: Client Unhappy with Design

**Response**:
"I want to make sure you're completely happy with the design. Can you share some examples of what you're envisioning? Remember, we have [X] rounds of revisions included. Let's schedule a call to discuss your feedback."

## Scaling Strategy

**When you have too many projects**:

1. **Raise prices** (increase RM 1,000-2,000)
2. **Hire freelancers**:
   - Designer: RM 1,000-2,000/project
   - Developer: RM 2,000-3,000/project
3. **Create templates** (reduce development time)
4. **Automate** (use tools for repetitive tasks)
5. **Specialize** (focus on one industry)

**Eventually**:
- Build a team
- Offer white-label services to agencies
- Create SaaS products
- Offer maintenance contracts for recurring revenue
