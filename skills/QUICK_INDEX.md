## 📞 常见问题 FAQ

### Q: 哪个 Skill 最重要？
A: **project-workflow** - 它统筹全局，确保项目顺利进行。

### Q: 我应该从哪个 Skill 开始学？
A: 
1. 先读 `project-workflow` 了解大局
2. 再学 `requirements-analyst` 掌握需求分析
3. 然后是 `design-consultant` 和 `frontend-builder`

### Q: 可以跳过某个 Skill 吗？
A: 不建议。4个 Skills 形成完整闭环，缺一不可。

### Q: 与 Claude Skills 有什么区别？
A: 
- **Kris Skills**: 业务流程和行业知识
- **Claude Skills**: 内容生成和工具能力
- **组合使用**: 效率翻倍！

### Q: 如何根据项目选择技术栈？
A:
- 简单宣传网站 → HTML + CSS
- 需要CMS → WordPress
- 复杂功能/高性能 → Next.js

### Q: 马来西亚本地化有什么特别？
A:
- 支付: 必须支持 FPX（本地银行）
- 语言: 至少 English + BM
- 节日: 考虑多元种族假期
- 物流: Ninja Van 和 J&T 最流行
